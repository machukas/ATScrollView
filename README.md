# ATScrollView

### Use

  - Create a view of type ATScrollView, or asign the class to a Storyboard view
  - Set up the class:
    ```swift 
    setHeaderView(_:) 
    ```
    ```swift 
    setContentView(_:) 
    ```
  - Configure
    ```swift 
    minHeaderHeight 
    ```
    ```swift 
    maxHeaderHeight
    ```
   - Optional
     ```swift 
     setScrollViewDelegate(_:)
     ```
